# 北京二手房数据分析
## 📌 项目背景
该项目使用Python爬虫从安居客收集二手房数据，
随后进行数据清洗和探索性数据分析，以研究房价模式。

## 🛠 技术栈
- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib

## 📊 数据字段
- title（标题）
- community（小区名称）
- area（地区）
- size（面积 单位：㎡）
- total_price(总价 单位：万)
- price_per_sqm(每平米价格 单位：万/㎡)

## 🚀 工作流程
1. 带分页控制的住房数据抓取
2. 清洗并预处理原始数据
3. 将文本字段转换为数值特征
4. 分析区域价格差异
5. 可视化价格分布与相关性

## 🔍 关键发现
- 东城、西城、海淀的房价相对其他区较高一些
- 房价集中在每平米3-4万左右
- 从此次分析的散点图可以看出总价和面积有着正相关趋势。
在整个分布范围内，点群大致围绕一条斜向上的直线分布，表明面积与总价之间存在近似的线性关系。
在相同面积上总价有较大的上下波动范围，说明面积不能完全的决定总价，有大量的其他因素在共同影响价格
